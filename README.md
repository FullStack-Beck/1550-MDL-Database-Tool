# 1550-MDL-Database-Tool
A revision-aware engineering data management system for MDLs, drawings, assemblies, and BOMs, designed to evolve into a Django-backed multi-user platform.

This project began as a frontend-only prototype to explore how engineering data — MDLs, drawings, parts, and assemblies — can be modeled and revised without the complexity of traditional PLM systems.

The core focus is on explicit revision control, hierarchical part structures, and carry-over workflows that preserve historical accuracy while allowing controlled evolution of data.

As the data model and workflows mature, the project will transition to a Django backend to support persistence, permissions, auditing, and collaboration.
